# 🚀 Featured Project

## NLP Intent Classification Engine

Transformer-based system that converts **natural language commands into spreadsheet operations.**

Example:

```
"Create a new column called profit and subtract expenses from revenue"
```

Pipeline:

```
User Query
   ↓
Intent Detection
   ↓
Expression Parser
   ↓
Operation Mapping
   ↓
Spreadsheet Transformation
```

Capabilities:

• Multi-intent classification  
• Expression parsing  
• Structured operation mapping  
• Automated data transformation
